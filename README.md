# The AI Capitol - Landing Page

A modern, mobile-optimized landing page for AI automation services built with Next.js 14.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
```

This will create a static export in the `out` directory that can be deployed to any static hosting service.

## Features

- Fully responsive design (mobile-first)
- Smooth scroll animations with Intersection Observer
- Animated counters for statistics
- FAQ accordion
- Modern gradient effects
- Touch-friendly tap targets

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- CSS Variables for theming

## Project Structure

```

├── public/            # Static assets

├── package.json       # Dependencies

```
